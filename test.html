    html常用标签简介。学习参考链接： http://www.w3school.com.cn/html/index.asp

    • 布局相关标签

      1. h1,h2,h3,h4,h5,h6标签:标题标签
      2. div标签:块元素标签
      3. span标签:被用来组合文档中的行内元素
      4. p标签:段落文本
      5. b标签:粗体文本
      6. i标签:斜体文本

    • 功能性标签

      1. a标签:超链接 页面跳转 新标签页打开
      2. img标签

    css基础样式介绍。学习参考链接： http://www.w3school.com.cn/css/index.asp

    • css基础样式

      1. height 属性 : 高度
      2. width 属性 : 高度
      3. margin 属性 : 外边距距离
      4. padding 属性 : 内边距距离
      5. border 属性 : 边框
      6. font 属性 : 字体定义
      7. background 属性 : 背景
